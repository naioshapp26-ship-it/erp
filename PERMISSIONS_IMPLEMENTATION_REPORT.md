## 🛠️ الدوال المساعدة

### 1. check_user_system_permission(user_id, system_code, required_level)
**الوصف:** التحقق من صلاحية المستخدم لنظام معين

**الاستخدام:**
```sql
SELECT check_user_system_permission(123, 'FINANCE', 'FULL');
```

### 2. check_user_approval_limit(user_id, amount)
**الوصف:** التحقق من صلاحية الموافقة المالية للمستخدم

**الاستخدام:**
```sql
SELECT check_user_approval_limit(123, 150000.00);
```

### 3. get_user_permissions_summary(user_id)
**الوصف:** عرض ملخص صلاحيات المستخدم الكامل

**الاستخدام:**
```sql
SELECT * FROM get_user_permissions_summary(123);
```

### مثال 2: الحصول على صلاحيات المستخدم
```javascript
router.get('/my-permissions', async (req, res) => {
    const userId = req.user.id;

        SELECT * FROM get_user_permissions_summary($1)
    `, [userId]);
    
    res.json(permissions.rows);
});
```
